# A compiler for lox
### 实现
- 1. 分配内存空间(malloc) 设计内存模型
- 2. 解析输入(scan) print实现
- 3. 构建字节码虚拟机
- 4. 反汇编字节码
- 5. 全局变量和局部变量
- 6. 分支与跳转(if else)
- 7. 循环(关键字 for while break continue)
- 8. 函数(关键字 func) 标准库(exit rand)
- 9. 闭包(closure) 嵌套函数
- 10. 垃圾回收 减少内存泄漏风险
- 11. 类(关键字 class)
- 12. 继承与多态
